# Frontend Boilerplate
a usefull boilerplate for your projects

## Contains
* sass
* pug
* browserify
* es6

## Setup

```
$ npm run setup
```

## Launch

```
$ npm start
```

## Clean

```
$ npm run clean
```

## Build For Development

```
$ npm run build:dev
```

## Build For Production

```
$ npm run build
```
